# Data Annotation Project for Amazon Product Reviews

## Objective
This project demonstrates a data annotation workflow for 
classifying Amazon product reviews into three sentiment 
categories: positive, negative, and neutral.

## Files
- `raw_reviews.csv` – original review dataset
- `annotated_reviews.csv` – manually labeled dataset
- `annotation_guidelines.md` – rules and edge cases for labeling
- `annotate.py` – Python script to support annotation and export

## Workflow
1. Created raw review dataset
2. Defined annotation rules and edge cases
3. Manually labeled reviews using the annotation guidelines
4. Reviewed labels for consistency and accuracy
5. Exported annotated data for downstream analysis

## Quality Checks
- Checked for missing labels
- Reviewed ambiguous cases using guidelines
- Verified consistency across positive, negative, and neutral
- Checked for duplicate records
